# Linux config files (Arch + Hyprland)

## Introduction

This directory holds the configuration for my https://www.archlinux.org/[Arch Linux]
machines: my shell (https://fishshell.com[fish], with https://www.zsh.org/[zsh] as a
fallback), my terminal (https://ghostty.org[ghostty]), and my Wayland compositor
(https://hyprland.org[Hyprland]) with the https://github.com/noctalia-dev/noctalia-shell[Noctalia]
bar/shell.

The full step-by-step is in the
https://github.com/Jubijub/arch-config/wiki[arch-config wiki].

Feel free to copy anything here, but use at your own risk.

## Deployment (chezmoi)

These files are managed with https://www.chezmoi.io[chezmoi], which handles both
deployment and the per-machine differences (workstation vs laptop). The source
layout uses chezmoi's conventions (`dot_config/` -> `~/.config`, `*.tmpl` for
templated files, `private_` for 0700 dirs). See
https://github.com/Jubijub/arch-config/wiki/5.Post-installation#deploy-dotfiles-with-chezmoi[the
chezmoi section of the wiki] for the bootstrap.

* `.chezmoi.toml.tmpl` asks once whether the machine is `workstation` or `laptop`
  and stores the answer; no hostnames are hardcoded.
* `hyprland.lua` and `noctalia/config.toml` are templates that branch on that role
  (monitors, GPU, battery/backlight).
* `system/` holds verbatim `/etc` drop-ins that chezmoi can't manage directly (it
  only touches `$HOME`); `.chezmoiscripts/run_onchange_install-system-files.sh.tmpl`
  installs them with `sudo` on `chezmoi apply`.

## Tools

### Themes
* Colors: https://github.com/catppuccin/catppuccin[Catppuccin Mocha]
* Font: https://www.jetbrains.com/lp/mono/[JetBrains Mono] / CaskaydiaCove, with the
  https://www.nerdfonts.com/[Nerd Fonts] patch
* GTK apps are forced dark via `gsettings ... color-scheme prefer-dark` (no theme
  engine -- full GTK theming was given up as too flaky).

### Shell & CLI
* https://fishshell.com[fish] : main shell, with https://github.com/jorgebucaran/fisher[Fisher]
  plugins and https://starship.rs[Starship].
* https://www.zsh.org[zsh] : POSIX-ish fallback, deliberately plugin-free.
* https://github.com/junegunn/fzf[fzf] : fuzzy history / file search.
* https://github.com/sharkdp/bat[bat] : `cat` with syntax highlighting.
* https://github.com/eza-community/eza[eza] : a modern `ls` replacement (colors, git-aware).
* https://github.com/sharkdp/fd[fd] and https://github.com/BurntSushi/ripgrep[ripgrep].
* https://docs.astral.sh/uv/[uv] : Python packaging / venv manager.
* https://wiki.archlinux.org/title/reflector[reflector] : picks the fastest Arch mirrors.

### Desktop
* https://ghostty.org[ghostty] : terminal (ships a corrected Catppuccin Mocha theme).
* https://github.com/noctalia-dev/noctalia-shell[Noctalia] : Wayland bar + shell.
* https://flameshot.org/[Flameshot] : screenshots with editing.

## Layout

`dot_config/` maps to `~/.config`, `dot_zshrc`/`dot_zshenv` to `~/.zshrc`/`~/.zshenv`,
`dot_local/` to `~/.local`. Highlights:

* `dot_config/hypr/hyprland.lua.tmpl` : Hyprland (machine-templated).
* `dot_config/noctalia/config.toml.tmpl` : Noctalia bar (machine-templated).
* `dot_config/ghostty/` : ghostty config + corrected Catppuccin Mocha theme.
* `dot_config/{bat,btop,fish,flameshot,tmux}/`, `dot_config/starship.toml` : CLI tools.
* `dot_config/nvim/` : Neovim config.
* `dot_config/pipewire/` : PipeWire tweaks.
* `dot_config/gtk-3.0/bookmarks` : Nautilus sidebar bookmarks.
* `dot_local/share/wallpapers/` : wallpapers.
* `system/etc/` : `/etc` drop-ins (locale, resolvconf, reflector, pacman hooks, and
  the Legion-only `90-hypr-gpu.rules`), installed by the run-script.
